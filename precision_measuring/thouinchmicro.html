Reading Hints

1. Sleeve scaleRead the last visible numbered tenth and any .025 marks showing to the left of the thimble edge.
2. Thimble scaleFind the thimble line that lines up with the sleeve reference line. Each thimble line is .001 inch.
3. Vernier scaleFor ten-thousandths micrometers, use the vernier line that matches best to add .0001 increments. This v1 uses thousandths; vernier mode comes next.
